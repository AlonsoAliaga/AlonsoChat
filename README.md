# AlonsoChat
This plugin format chat allowing you to use hover and click actions per format.\

### ⚠️ **This plugin requires [AlonsoLib](https://alonsoaliaga.com/AlonsoLib) installed.** ⚠️
### ⚠️ [AlonsoLib](https://alonsoaliaga.com/AlonsoLib) is only available on GitHub, that's why AlonsoChat can't be installed in Aternos. ⚠️
